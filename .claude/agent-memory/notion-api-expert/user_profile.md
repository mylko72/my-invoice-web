## 기술 환경
- Next.js 16.1.6 (App Router)
- React 19.2.3
- TypeScript 5
- Tailwind CSS v4
- shadcn/ui 컴포넌트

## 프로젝트 특성
- 인보이스 웹 애플리케이션 (invoice-web)
- 한국어 문서화, 주석, 커밋 메시지 선호
- 2칸 들여쓰기 코드 스타일
- camelCase 함수/변수, PascalCase 컴포넌트

## 언어 규칙
- 응답: 한국어
- 코드 주석: 한국어
- 변수/함수명: 영어
- 커밋 메시지: 한국어
